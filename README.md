## 🛠️ Tecnologías Utilizadas

Para el desarrollo de este portafolio se utilizaron tecnologías modernas y eficientes:

* **HTML5:** Estructura semántica del sitio.
* **Tailwind CSS (v3):** Framework de CSS utilizado para todo el diseño visual, espaciado, tipografía y adaptabilidad (responsividad). Se cargó mediante CDN para prototipado rápido.
* **JavaScript (Vanilla):** Utilizado para la funcionalidad del formulario de contacto y animaciones sutiles.
* **Font Awesome (v6.4.0):** Biblioteca de iconos vectoriales para mejorar la interfaz visual.
* **Google Fonts:** Tipografía "Plus Jakarta Sans" para una lectura clara y moderna.
* **Formspree:** Servicio utilizado para gestionar el envío de correos desde el formulario de contacto sin necesidad de backend propio.

---

## ⚙️ Características Destacadas

1.  **Diseño Responsivo:** Se adapta perfectamente a dispositivos móviles, tablets y computadoras de escritorio.
2.  **Video de Fondo (Hero):** Un video en bucle ajustado y responsive que le da un toque dinámico y temático a la introducción.
3.  **Estilo Glassmorphism:** Uso de fondos semi-transparentes con desenfoque (`backdrop-filter: blur`) para un aspecto moderno y limpio.
4.  **Efectos Hover:** Tarjetas de habilidades y proyectos con animaciones suaves y sombras de neón al pasar el mouse.
5.  **Formulario Funcional:** Integración directa con Formspree para recibir mensajes directamente en mi correo electrónico.
6.  **Optimización de Legibilidad:** Uso de opacidad en el video de fondo y fondos oscuros (`slate-950`) para garantizar que el texto sea siempre legible.

## 🔧 Uso y Despliegue Local

Si deseas clonar este repositorio y ejecutarlo localmente:

1.  Clona el repositorio: `git clone https://github.com/lacroux/mi-portafolio.git`
2.  Navega a la carpeta: `cd mi-portafolio`
3.  Abre el archivo `index.html` en tu navegador web preferido.

*Nota: Para que el formulario de contacto funcione correctamente, deberás configurar tu propio ID de Formspree en el atributo `action` del formulario en el archivo HTML.*
